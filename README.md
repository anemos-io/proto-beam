# Proto-Beam

## Description

Utilities for handling and converting ProtocolBuffers. Focused but not
limited to Apache Beam.

## Roadmap

- ProtoBuf to BigQuery DDL
- ProtoBuf to Spanner
- ProtoBuf to/from BSON
- ProtoBuf to/from Avro
- Recursive messages
- OneOf type
- Well Known Types
- Wrapper Types
- Plugable message type convertions
- Options support for
    - Description
    - Deprecation
    - Flattening
